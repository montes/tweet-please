Tweet Please is a plugin that let's you have auto-tweeted all your new posts.

== Description ==

Get your posts' link auto-posted to twitter with this features:

* Write your own tweet text
* Select when to publish (inmediately, after 1 hour, after 2 hours, next 10am...)
* Each posted tweet will appear on the posted tweets log (at Options > Tweet Please)
* It really works! tweets appear on twitter!

You can contribute to this plugin at: https://github.com/montes/tweet-please/

== Installation ==

1. Upload `tweet-please` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Fill options at Settings > Tweet Please
